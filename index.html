Who am I?

My name is Donald McKercher, but everyone just calls me Donny. If you've found this page you've probably guessed that I'm a Web Developer and want to learn a little more about me.

What Kind of work do I do?

I mostly work with Front-end development such as: HTML5, CSS3(SASS/SCSS), JavaScript, and JQuery. I do some Back-end development previously I worked with PHP and MySQL, but I've been learning Node.js, Express.js, and MongoDB and I'm really enjoying it!

On the design side I do some wireframing with Sketch 3

How I do my work

Well it all starts with the planning process. First I need to assess the bare essentials for the project. Like the purpose of the project, who's the target audiance, what fuctionality the site needs, and what tools are needed for the project. From there I can develop the attack plans! I then move onto the design to determine what I'm working with. I'll wireframe something together with Sketch, get some feedback maybe make some revisions and move on. Now I can start doing some coding. After reciving some final feedback make some more revisions per the customers requests.

Where do I want my career to go?

I really enjoy writing Node. I'd like to get into writing apps with Node and JavaScript
